Papierkram Dark Mode – Chrome Extension

This privacy policy describes how the "Papierkram Dark Mode" browser extension handles user data.

Data Collection

This extension does not collect, store, transmit, or share any personal data, browsing history, or any other user information.

Data Storage

The extension stores a single preference locally on your device using Chrome's built-in storage API (chrome.storage.local): whether dark mode is enabled or disabled. This data never leaves your browser and is not accessible to anyone other than you.

Third-Party Services

This extension does not use any third-party services, analytics, tracking tools, cookies, or external servers. No network requests are made by the extension.

Permissions

The extension requests the following permissions:

  • Storage: To save your dark mode toggle preference locally on your device.
  • Content Script access to papierkram.de: To inject the dark mode CSS styles into papierkram.de pages. The extension only runs on papierkram.de and its subdomains.

Affiliation

This extension is not affiliated with, endorsed by, or connected to Papierkram or its parent company in any way. It is an independent, community-created tool.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date.

Contact

If you have any questions about this privacy policy, you can reach out via the Chrome Web Store listing or the extension's support page.
